# 柚子软件库

一个功能完整的柚子云应用商店，基于纯HTML/CSS/JavaScript开发，提供专业的软件技术交流平台。

## ✨ 功能特性

- **响应式设计** - 完美适配手机、平板、电脑三端
- **现代UI设计** - Material Design风格，渐变色设计
- **完整功能** - 搜索、分类筛选、分页加载、下载等完整功能
- **错误处理** - 完善的错误边界和用户友好提示
- **SEO优化** - 完整的元标签和Open Graph支持
- **高性能** - 防抖搜索，异步加载，优化用户体验

## 🚀 快速开始

### 本地预览

1. 下载项目文件
2. 使用任何HTTP服务器打开 `index.html`
3. 或直接双击 `index.html` 在浏览器中查看

### 功能演示

- **搜索功能**: 在顶部搜索框输入关键词，实时搜索应用
- **分类筛选**: 点击"全部"或"AE工程"按钮筛选应用
- **分页加载**: 点击"加载更多"按钮获取更多应用
- **下载应用**: 点击应用卡片弹出模态窗口，支持iframe内嵌下载
- **弹窗交互**: ESC键关闭弹窗，点击遮罩层关闭

### 部署选项

项目支持多种部署方式：
- **Cloudflare Pages**: 零配置部署
- **Netlify**: 简单的拖拽部署
- **GitHub Pages**: 免费静态网站托管

## 📱 设备适配

- **手机端**: 单列布局，底部导航，触摸友好的交互
- **平板端**: 双列布局，优化触摸体验
- **桌面端**: 多列布局，顶部导航栏，悬停效果

## 🎨 界面设计

- **Material Design风格**: 现代化的卡片式布局，渐变色设计
- **柚子云风格**: 深色背景，醒目的应用图标

- **HTML5**: 语义化标签，现代Web标准

- 构建设置：构建命令留空，输出目录为`/`

#### 2. Netlify
- 直接拖拽项目文件夹到Netlify
- 自动检测静态网站配置

#### 3. GitHub Pages
- 推送到GitHub仓库
- 在仓库设置中启用GitHub Pages
